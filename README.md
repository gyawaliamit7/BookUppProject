# BookUppProject

### BookUpp- A Crowdsourced Library Book Sharing App
## Authors

 Amit Gyawali
 
 Ashutosh Dahal
 
 Kshitiz Hamal
 
 Kushal Singh

## Description

BookUPp is a an Android Application that allows students to borrow and lend

book within a local community. 

## Features

  -NFC(Near Field Communication) Login
  
  -ISBN Scanner
